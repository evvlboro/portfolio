# Сайт-визитка

Пока искал работу, стало скучно — и я решил сделать сайт-визитку, чтобы прикрепить ссылку на него в резюме.

**Сайт:** [https://portfolio-aprilrynn.vercel.app/](https://portfolio-aprilrynn.vercel.app/)

## Стек

- [Next.js](https://nextjs.org) 16 (App Router)
- [React](https://react.dev) 19
- [TypeScript](https://www.typescriptlang.org)
- [Tailwind CSS](https://tailwindcss.com) v4
- Форма обратной связи: Telegram Bot API + Google reCAPTCHA v2

При создании проекта использовал [Cursor](https://cursor.com).

## Демо в портфолио
